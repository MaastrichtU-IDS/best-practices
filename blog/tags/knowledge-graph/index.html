Build a FAIR Knowledge Graph

March 18, 2021 Â· 10 min read

Vincent Emonet

Data Science engineer at IDS

This short tutorial guides you through building a RDF Knowledge Graph about restaurants and cuisines from 2 CSV sample files generated from a dataset found on Kaggle, using mappings expressed in YAML.

We will use tools, data models and formats based on recommendations from the World Wide Web Consortium (W3C) to build a knowledge graph as FAIR as possible (Findable, Accessible, Interoperable, Reusable).

The instructions and tools are accessible to anyone without the need to understand anything else than reading English, just follow the instructions and copy/paste the text at the right place in a website, and build a knowledge graph!

This tutorial will introduce what standard knowledge graphs are, and guide you to:

  • Search for relevant ontologies/vocabularies to describe your knowledge graph content
  • Properly deal with using multiple ontologies, and describing missing properties
  • Map 2 CSV files to a coherent RDF knowledge graph, and generate the RDF data
