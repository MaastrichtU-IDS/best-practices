Continuous integration

Checkout the workshop

See our IDS CI/CD workshop to package and define Continuous Integration workflow for Python with GitHub Actions.

The choice of CI/CD technology depends on the platform you are using to store your code

  • GitHub Actions
    • Recommended if your code is hosted on GitHub
  • GitLab CI
    • Recommended if your code is hosted on GitLab.
    • We recommend to use GitLab hosted at UM if possible.
  • Jenkins
    • Recommended if your build is privacy sensitive.
