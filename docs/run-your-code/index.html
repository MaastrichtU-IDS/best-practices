Where to run your code?

On your machine

Installed locally

Better for development, to use development and debug environments.

Using Docker

Better when just running an application without doing development, or when running in production on a server.

On the DSRI

The Data Science Research Infrastructure at Maastricht University provides CPU and GPUs nodes for Data Science computing, requires a VPN connection to the UM network.

On Google Collab

Run Jupyter Notebooks on Google servers for research.

On Kaggle

To run Jupyter Notebooks when participating to a Kaggle Data Science competition.

Free during the competition.

Using CI/CD solutions

GitHub Actions or GitLab CI to run workflows on their servers triggered by event such as git push, new release, new pull request...

Usually used to run tests.

Free for Open Source
